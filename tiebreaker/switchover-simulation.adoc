= Simule uma transição usando o desempate do MetroCluster.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
A partir da versão 1.7 do MetroCluster Tiebreaker, é possível simular uma troca de controle para testar a funcionalidade de troca de controle do Tiebreaker.

.Sobre esta tarefa
* O `monitor switchover-simulate` O comando só é compatível com o Tiebreaker 1.7 ou posterior.


.Passos
. Verifique o status de monitoramento do Tiebreaker:
`monitor show -status`

. Realizar uma simulação de transição:
+
`monitor switchover-simulate -monitor-name <monitor_name> -cluster <cluster_name>`

+
O comando retorna a seguinte saída para uma operação bem-sucedida:
