= MetroCluster fluxo de trabalho de configuração de desempate
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Após instalar o software Tiebreaker, você pode adicionar configurações do MetroCluster, configurar as definições de SNMP e começar a monitorar seu ambiente MetroCluster.

.Sobre este fluxo de trabalho
Você pode usar este fluxo de trabalho para configurar o software Tiebreaker após a instalação.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Um"]link:concept_configuring_the_tiebreaker_software.html["Configure o software tiebreaker"]
[role="quick-margin-para"]
Adicione configurações do MetroCluster ao software Tiebreaker, importe certificados e configure as configurações de monitoramento.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Dois"]link:task_configure_snmp_setts_for_tiebreaker_software.html["Configurar as configurações SNMP para o software MetroCluster Tiebreaker"]
[role="quick-margin-para"]
Configure as definições de SNMP para que o software Tiebreaker possa enviar notificações SNMP para eventos do MetroCluster.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Três"]link:concept_monitoring_the_mcc_configuration.html["Monitore a configuração do MetroCluster"]
[role="quick-margin-para"]
Utilize o software Tiebreaker para monitorar o status da configuração do MetroCluster, visualizar eventos SNMP e gerenciar as configurações do AutoSupport.
